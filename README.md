# Калькулятор матриц на Go

## Описание
Данное консольное приложение позволяет выполнять операции над матрицами размером 2x2 или 3x3:
- Сложение матриц
- Умножение матрицы на число
- Умножение двух матриц друг на друга

Программа разделена на функции для выполнения каждого из указанных видов операций, что обеспечивает модульность и удобство поддержки.

## Как запустить
1. Склонируйте репозиторий или скопируйте код в файл, например, `main.go`.
2. Откройте терминал и перейдите в каталог с файлом.
3. Выполните команду:
```bash
go run main.go
