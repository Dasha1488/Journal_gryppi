# Журнал группы на Go

## Описание
Данное консольное приложение позволяет вести учет успеваемости студентов группы. Основные функции:
- Добавление информации о студенте (ФИО, оценки)
- Просмотр всех студентов
- Фильтрация студентов по среднему баллу (например, вывод студентов с оценками ниже 4)
- Подсчет среднего балла каждого студента

Программа использует структуру `Student`, карту (`map`) для хранения данных и слайсы для хранения оценок.

## Как запустить
1. Создайте файл `main.go` и вставьте туда приведённый код.
2. В терминале перейдите в директорию с файлом.
3. Выполните команду:
go run main.go
